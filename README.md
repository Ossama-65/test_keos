# Dashboard Prospection

Dashboard de gestion de prospection avec génération automatique et tracking.

## Configuration

1. Créer un fichier `.env.local` :

```env
AUTH_PASSWORD=votre_mot_de_passe_securise
```

2. Installer les dépendances :

```bash
npm install
```

3. Lancer le serveur de développement :

```bash
npm run dev
```

4. Accéder à l'application : http://localhost:3000

## Connexion

- Mot de passe par défaut : `admin123`
- Modifiez `AUTH_PASSWORD` dans `.env.local` pour un mot de passe sécurisé

## Fonctionnalités

- Dashboard avec KPIs
- Liste et détail des prospects
- Génération automatique de prospects
- Enrichissement des données
- Gestion de campagnes
- 100% responsive (mobile, tablet, desktop)
